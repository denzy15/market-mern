# Интернет магазин на Mongo+Express+React+Node 
## Основные страницы: 
1. Глваная
2. Поиск и сортировка по фильтрам
3. Корзина
4. Редактирование профиля
5. Оформление заказа
6. История покупок
7. Написание отзыва (Доступна если товар был "куплен")
